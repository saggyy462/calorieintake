# Changelog

## v0.6.0 - 2026-02-12

- Added historical date workflow:
  - `Log Date`, `Prev Day`, `Next Day`, `Today`
  - full add/edit/delete/clear on selected date context
  - today vs historical view badge
- Improved compact mode with slot expansion and graph-click expansion.
- Added meal-slot macro graph for quick visual comparison.

## v0.5.0 - 2026-02-12

- Added compact mode toggle (persisted in local storage).
- Reworked meal flow cards with macro-forward UI.
- Added quick action dropdown replacing multiple action buttons.

## v0.4.0 - 2026-02-12

- Added strict planner checks to reduce macro overshoot.
- Added editable logged meals UI (edit/delete/clear slot).
- Added `Save Unparsed Item` custom database flow.
- Added scalable custom per-gram food handling.

## v0.3.0 - 2026-02-12

- Added day-specific targets (7-day model).
- Added slot-based meal logging.
- Added suggestion engine (slot suggestion + remaining day plan).
- Added projected gap against targets.

## v0.2.0 - 2026-02-12

- Expanded South-Indian food defaults and parsing aliases.
- Added support for grams/cups/dl/count inputs.
- Added PWA metadata and iOS deployment guide.

## v0.1.0 - 2026-02-11

- Initial app scaffold:
  - targets
  - plain-text meal logging
  - totals and remaining macros
  - local storage persistence
